РАБОТА С КОММУТИРУЕМЫМИ КАНАЛАМИ СВЯЗИ

1. Настройка удаленного доступа

Для организации передачи данных через модем по протоколам TCP/IP в ОС Windows 9x необходимо выполнить следующие действия:

  1. Убедиться, установлен ли модем в системе, — если нет, то выполнить его установку и настройку его параметров.
  2. Выполнить те же действия для контроллера удаленного доступа и службы удаленного доступа к сети. После этого конфигурация сети будет выглядеть аналогично рис. 1.

Рис. 1. Конфигурация сети
  1. В параметрах привязки сетевой карты к протоколам
    (    Control Panel/Network/NE2000 Adapter/Properties/Bindings) снять флажок, соответствующий протоколам TCP/IP     (см. рис. 2.).

Рис. 2. Привязка сетевой карты к протоколам
  1. В параметрах привязки контроллера удаленного доступа к протоколам (Control Panel/Network/Dial-Up Adapter/Properties/Bindings) установить флажок, соответствующий протоколам TCP/IP (см. рис. 3.).

Эти действия выполняются для того, чтобы разрешить конфликт сетевой карты и контроллера удаленного доступа при их одновременной работе в системе.

Остальные флажки, соответствующие другим сетевым протоколам (если эти протоколы используются в работе) необходимо установить аналогично описанию в пунктах 3 и 4.

2. Установка модемного соединения

Для установки модемного соединения необходимо осуществить следующие действия:

  1. Включить модем и убедиться в его функционировании с помощью средств диагностики модема.
  2. Осуществить настройку используемого соединения (при необходимости создать его). Используемые соединения доступны через My Computer/Dial-Up Networking (см. рис. 4). При этом необходимо указать телефонный номер, по которому модем будет осуществлять дозвон. В данном случае это двузначный номер мини-АТС, закрепленный за ПЭВМ, с которой устанавливается связь. Настройку соединений необходимо осуществить и для клиента, и для сервера (см. рис. 5.). Кроме того, для сервера нужно убедиться, что установлен сервер удаленного доступа и для него разрешен ответ на все входящие звонки.
