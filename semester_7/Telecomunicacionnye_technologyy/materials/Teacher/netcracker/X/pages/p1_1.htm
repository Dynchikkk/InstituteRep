1. Знакомство с NetCracker Professional.

Цель занятия:

·        познакомиться с графическим интерфейсом пользователя (GUI).;
·        определить все элементы главного окна приложения NetCracker Professiona;

·        изучить средства и способы доступа к инструментам и режимам.

Запустите пакет NetCracker Professional из меню Пуск рабочего окна Windows.
 

Откроется главное окно приложения (рис.1.1). Помимо главного меню и приборных панелей главное окно NetCracker Professional содержит 3 панели: браузер, рабочую область и панель изображения. Когда вы начинаете работу рабочая область содержит пустой сайт Net1. Панель изображений заполнена изображениями устройств и приложений выбранного входа базы данных.
 



Рис.1.1 - Главное окно NetCracker Professional.
 

Откройте файл NetCracker Professional (.NET). Для этого в меню File выберите Open. Появится диалоговое окно открытия файла (см. рис 1.2).

Рис.1.6 - Диалог конфигурации.
 
  • чтобы выбрать процессор HSSI интерфейса, на панели выбора, ниже имени прибора нажмите на первом значке.
     
  • теперь выберите Each slot на изображении устройства. При выборе каждой части он высвечивается и на изображении устройства, и на панели выбора.
     
  • для доступа к информации об этой части используйте один из методов.

a) на панели выбора кликните TAXI процессора АТМ интерфейса,
    нажмите правую кнопку и выберите меню Свойства.
b) на панели выбора кликните TAXI процессора АТМ интерфейса, нажмите кнопку Setup.
c) на изображении устройства кликните TAXI процессора АТМ интерфейса, нажмите кнопку Setup.



Рис.1.7 - Окно настройки встроенного устройства.
 
  • в свойствах выберите вкладку Protocol для того, чтобы видеть разрешенные протоколы,
     
  • чтобы закрыть диалог, нажмите Cancel или OK. Теперь вы снова в диалоге конфигурации,
     
  • чтобы увидеть, как устроен Cisco 7000, нажмите кнопку установки устройства и выберите вкладку
    Port, чтобы видеть, сколько портов используется и сколько - нет, затем закройте диалог Свойства.

Замечание: не закрывать окно конфигурации!

  • далее вставим другую часть в устройство. Нажмите недавно используемые вкладки на панели изображения, спускайтесь по ней до ATM Interface Processor, DS3. Нажмите на ATM Interface Processor, чтобы выбрать его, выделите его левой кнопкой мыши и тяните в диалог конфигурации
    до тех пор, пока мышь не будет над открытой щелью в изображении устройства, затем отпустите
    кнопку мыши.

помещена (). Если вы перетаскиваете его к открытому слоту курсор меняется на ,
