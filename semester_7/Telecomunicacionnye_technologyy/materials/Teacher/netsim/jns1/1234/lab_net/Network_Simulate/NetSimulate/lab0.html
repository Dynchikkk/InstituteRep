Введение.

Основная цель предлагаемого вашему вниманию материала - дать его читателям представление об основах организации IP-сетей на сетевом, транспортном и пракладном уровнях, привить начальные навыки по настройке, моделированию и исследованию небольших ЛВС, познакомить с основами адресации и статической маршрутизации в ЛВС.

Эти сведения могут быть полезны студентам, обучающимся по направлению 230400.62 "Информационные системы и технологии" в рамках курса "Инфокоммуникационные системы и сети", а также студентам других специальностей, изучающих предметы "Компьютерные сети", "Вычислительные машины, системы и сети" и "Сетевые технологии".

Данный материал представляет собой сокращенную html-версию пособия "Исследование уровней организации IP-сетей. Лабораторный практикум", выпущенного в СПбГЭТУ "ЛЭТИ" авторским коллективом в составе: Алекперов И.А., Большев А.К., Карпов К.Э., Кринкин К.В. и Яновский В.В. Для целей наглядной демонстрации функционирования стека TCP/IP, на кафедре МО ЭВМ был разработан эмулятор javaNetSim. В качестве исходного варианта авторы использовали разработку Канберрского технического университета JFirewallSim, распространяемую по лицензии BSD.

Эмулятор javaNetSim имитирует работу протоколов каждого из уровней, что позволяет моделировать и исследовать различные структуры организации сети. Обладая достаточной функциональностью, вместе с тем он очень прост в работе и удобен для выполнения студентами лабораторных работ.

Из комерческих программных продуктов, наиболее известен Packet Tracer — эмулятор сети передачи данных, выпускаемый фирмой Cisco Systems. Он позволяет делать работоспособные модели сети, настраивать маршрутизаторы и коммутаторы, взаимодействовать нескольким пользователям. Кроме того, он содержит серверы DHCP, HTTP, TFTP, FTP, рабочие станции, устройства WiFi, различные кабели. Все это позволяет успешно создавать даже сложные макеты сетей, проверять на работоспособность топологии. Но этот программный продукт доступен бесплатно только для участников Программы Сетевой Академии Cisco.

Как пишет один из авторов проекта, эмулятор сетей javaNetSim - это попытка сделать "опенсорсный" PacketTracer. Конечно, до PacketTracer ему далеко, но для начального обучения студентов основам сетевых технологий вполне достаточно.

Приводимое ниже описание лабораторных работ базируется на версии javaNetSim-0.33. В настоящее время в Интернете на сайте http://sourceforge.net доступна версии javaNetSim-0.41. Основные отличия версии 0.41 (страничка проекта) от предыдущего открытого релиза 0.33 состоят в следующем:

  • Добавлены Wi-Fi AP и лэптопы, реализован в упрощенном виде 802.11b, эмулируется WEP-шифрование.
  • VLANы и свичи 2-го уровня.
  • ACL и NATы работают в полном объеме, консоль в рутерах стала еще больше похожа на Cisco.
  • OSPF.
  • DNS запросы теперь доступны и из консоли тоже.

Имитатор javaNetSim является объектно-ориентированным и написан на языке Java. Программы, написанные на этом языке, являются машинно-независимыми. То есть имитатор javaNetSim будет работать на любом компьютере, для которого есть виртуальная Java машина. Если на вашем Windows-компьютере отсутствует поддержка Java, то вы можете обратиться на сайт http://java.com/ru/

Файл javaNetSim.jar это Java-архив, который представляет собой обычный ZIP-архив, в котором содержится часть программы на языке Java. Команда для запуска имитатора javaNetSim имеет вид:

java -jar javaNetSim.jar
При загрузке пакета из интернета в его составе обычно присутствует файл
windows.bat ,
который, являясь командным файлом, служит для запуска javaNetSim из среды Windows.

Хотя язык Java является интерпретируемым, это не оказывает существенного влияния на быстродействие имитатора. Это объясняется тем, что имитатор разрабатывался для моделирования работы небольших сетей, обработка моделей которых не требует больших вычислительных ресурсов.

Подводя итог, следует отметить, что javaNetSim - зто доступный эмулятор, отвечающей требованиям учебного процесса: минимальным временным затратам на обучение, комплексным требованиям к аппаратным платформам, инвариантностью к операционным системам.

Малые объемы требуемой памяти, хорошие временные показатели, многоплатформенность, наглядный графический интерфейс, обеспечивают javaNetSim преимущества по сравнению с другими эмуляторами.
