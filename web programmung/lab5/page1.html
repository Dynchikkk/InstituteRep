Название

Разработка документа сложной структуры с использованием языка HTML

Цель работы

Изучение элементов HTML управления цветом и вставки объектов в документ.

Задание

Разработать HTML программу для вывода на Web страницу отчета о лабораторной работе

Порядок выполнения работы

  • Изучить особенности использования языка HTML, приведенные в документах "Основные понятия ", "Элемент ", "Элемент BODY"
  • Изучить принципы управления цветом на Web страницах
  • Изучить принципы вставки объектов на Web страницы
  • Изучить принципы макетирования документа с помощью CSS
  • Разработать программу в соответствии с заданием на нее
  • Составить отчет о выполнении лабораторной работы, включив в него стандартные разделы
