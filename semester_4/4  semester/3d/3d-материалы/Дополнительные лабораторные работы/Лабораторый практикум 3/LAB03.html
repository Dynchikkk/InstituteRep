Теоретическая часть

Для создания изображений ЗD-объектов в САПР AutoCAD предусмотрено подмножество команд, выполняемых в определенной последовательности. Рассмотрим объект проектирования рис .1., состоящий из одного тела - прямоугольной призмы. Он может быть синтезирован альтернативными способами.

Рис.1. Объемный объект проектиования.

Способ 1 - задание тела толщиной - включает в себя следующие шаги:

1. Для ввода описания модели тела, Ннижняя грань которого находится на плоскости Z =2.0, а верхняя - на плоскости Z = 4.0, используется команда ELEV <ENTER> (от англ. elevate - поднимать),

Опциями команды являются:

- New current elevation <0.0000>: 2.0
(Новый текущий уровень подъема основания тела)

- New current thickness <0.0000>: 2.0
(Новая текущая толщина тела)

2. С помощью команды SOLID <ENTER> описывается сплошной опорный контур тела, а именно по запросам команды вводятся мировые (X,Y) - координаты точек его проекции на плоскость Z = 0.0:

  1. First point! 2.0, 2.0 <ENTER>    (первая точка)
  2. Second point: 2.0, 7.0 <ENTER>   (Вторая -точка)
  3. Third point: 8.0, 2.0 <ENTER>    (третья точка)
  4. Fourth point: 6.0, 7.0 <ENTER>   (четвертая точка)
  5. Third point : <Ctrl-C> <Cancel> (ввод закончен)

3. Аналогичным образом с помощью команд ELEV и SOLID вводятся описания других тел, если они содержатся в объекте.

4. С помощью  команды VPOINT <ENTER> (ввод точки наблюдения) выполняются необходимые видовые и экранные аффинные преобразования описанных тел для создания их ортогональных иди аксонометрических проекций. Команда вводится с клавиатуры иди выбором иэ падающего меню.

Опция команды: Enter view point <0.0000,0.0000,0.0000>: (запрашиваются мировые координаты точки наблюдения).

После выполнения команды на экране воспроизводится скелетная (проволочная) модель объекта.

5. С помощью команды HIDE <ENTER> выполняется удаление невидимых линий и поверхностей, и на экране воспроизводится твердотельная непрозрачная модель объекта.

Все команды простановки размеров и нанесения надписей, команды редактирования выполняются до VPOINT, так как последняя выполняется совместно с HIDE. В том случае, если тело имеет полость, оно представляется совокупностью соприкасающихся тел рис .2. - прямоугольных паралепипедов. Соединения (склеивания) их в одно тело при нулевых зазорах не происходит, т.е. на проекциях видны линии границ тел.

Рис.2. Представление тела с дефектом.

Способ 2 - задания границ тела - включает следующие шаги:

1.С помощью команды 3DFACE (трехмерная грань) описывается каждая из множества плоских граней тела. Команда 3DFACE аналогична команде SOLID, но при вводе запрашиваются не 2, а 3 координаты вершин грани (3Dpoint - 3D точка).

Формат команды следующий: 3DFACE <Enter>

  • First Points ( 3Dpoint )         (первая точка грани)
  • Second Points ( 3Dpoint )        (вторая точка)
  • Third Point ( 3Dpoint )         (третья точка)
  • Fourth Point ( 3Dpoint or <Enter>)  (четвертая точка или <Enter>для замыкания треугольной грани)
  • Third Point:(3Dpoint or<Enter>)     (точка или <Enter>,для завершения ввода)

Для получения твердотельной модели необходимо вводить вершины каждой из граней, обходя их по часовой стрелке при наблюдении граня снаружи.

2. Командой VPOINT задается точка наблюдения тела и выполняется видовое преобразование тела.

3. Командой HIDE удаляют невидимые части изображения твердотельной модели объекта.

Экспериментальная часть

  1. На основании исходных данных для AutoCAD выполнить ЗD-моделирование объектов сплошными  твердотельными моделями с помощью необходимого подмножества команд описанными выше способами.
  2. Получить и распечатать изображение объекта в необходимых проекциях.
  3. Для изучения возможностей AutoCAD по автоматическому синтезу объемных тел выбрать в падающем командном меню пункт DRAW (рисовать), а в его подменю - пункт 3D-CONSTRUCTION, и, пользуясь графическим меню (так называемой иконой), на котором показаны фреймы тел, выбрать поочередно различные конструкций. Настройка осуществляется вводом параметров с клавиатуры (число граней при аппроксимации поверхностей) и с помощью манипулятора "мышь" (ввод координат точек). Для просмотра тел использовать команду VPOINT.ив программу, распечатать ее текст на принтере.
